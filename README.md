# 👻 Haunted Manor 3D

A first-person **horror exploration game** set inside a mysterious haunted mansion.  
Your mission: **find the cursed book** hidden somewhere in the manor before time runs out...  
But be careful — **ghosts, shadows, and strange sounds** roam the halls. The book isn’t always where you left it 👀  

---

## 🕹️ Gameplay
- Explore a dark and eerie 3D mansion.  
- Search for the ancient book before the timer ends.  
- Ghosts will appear randomly to scare and distract you.  
- When you finally spot the book... it might **disappear** and reappear somewhere else.  
- Every playthrough is slightly different — expect the unexpected.

---

## 🎮 Features
- Realistic 3D environment built with Unity.  
- Dynamic lighting and ambient horror sound design.  
- Randomized ghost encounters and book locations.  
- Countdown timer adds pressure to every moment.  
- Immersive atmosphere inspired by classic horror games.
